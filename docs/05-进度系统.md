## 3. 关卡系统

### 3.1 关卡结构

| 关卡 | 时长 | 难度 | 敌人类型 |
|------|------|------|----------|
| 1 | 2分钟 | 1.0x | 被动型为主 |
| 2 | 2分钟 | 1.2x | 被动型+冲撞型 |
| 3 | 2分钟 | 1.5x | 全部基础类型 |
| 4-5 | 2分钟 | 1.8x | 追踪型首次出现 |
| 6-10 | 2分钟 | 2.0x+ | 含精英怪 |
| 11+ | 2分钟 | 3.0x+ | 高难混合 |

### 3.2 难度曲线

```
难度系数 = 1.10 ^ (关卡 - 1)

关卡1:  1.00x  HP30   伤害10
关卡10: 2.14x  HP64   伤害16
关卡20: 6.73x  HP202  伤害33
关卡30: 17.4x  HP524  伤害62
关卡50: 97x    HP2914 伤害156
```

### 3.3 关卡进度

- 怪物血量：每关 +10%
- 怪物伤害：每关 +5%
- 生成频率：每关 -5%间隔
- 精英怪出现率：每关 +2%
