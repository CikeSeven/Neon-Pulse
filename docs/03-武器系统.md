## 1. 武器概述

### 1.1 核心机制

```
自动射击：武器自动按射速发射，无需手动操作
环绕排列：武器围绕玩家自动排列
升级成长：武器可独立升级，伤害/射速/特效提升
```

### 1.2 武器属性

| 属性 | 说明 |
|------|------|
| 伤害 | 每次攻击造成的伤害值 |
| 射速 | 每秒发射次数 |
| 穿透 | 可穿透的敌人数量 |
| 散射 | 子弹分散角度 |
| 弹速 | 子弹飞行速度 |

## 4. 武器管理器

### 4.1 功能说明

```
武器管理器职责：
  • 管理玩家拥有的所有武器
  • 排列武器在玩家周围
  • 处理武器切换
  • 应用武器升级效果
```

## 5. 武器升级

### 5.1 升级效果

| 属性 | 每级成长 |
|------|----------|
| 伤害 | +15% |
| 射速 | +5% |
| 穿透 | 每3级+1 |
| 特殊 | 因武器而异 |

### 5.2 武器等级成长示例

```
Neon AK 伤害成长：
  LV.1: 6
  LV.3: 8 (×1.15²)
  LV.5: 10 (×1.15⁴)
  LV.10: 24 (×1.15⁹)

霰弹枪弹丸数成长：
  LV.1-2: 5
  LV.3-4: 6
  LV.5-6: 7
  每2级+1弹丸
```

### 5.3 升级费用

| 等级 | 价格系数 |
|------|----------|
| 1→2 | 50 |
| 2→3 | 60 |
| 3→4 | 72 |
| ... | 每级×1.2 |
| 9→10 | 约 150 |
